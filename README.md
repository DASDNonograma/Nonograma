# Nonograma
Proyecto de Diseño Avanzado de Sistemas Digitales. Es el diseño de un puzzle llamado nonograma en VHDL.

## Estructura del repositorio

**Diagrama Disenio**: Contiene los diagramas de las unidades de control y proceso creados con draw.io (ahora llamado diagrams.net). Tiene dos subdirectorios, *FicherosDrawio* con los ficheros para abrirlos con el diagrama y *DisenioExports* con los diagramas exportados en formato .pdf o png.

**FicherosVHDL**: Proyeto Modelsim con ficheros fuente de los diseños VHDL, ficheros de simulación para realizar el testbench y scripts do para formatear los datos de simualción cómodamente.

**Quartus**: Proyecto Quartus Prime con fichero TOP y configuración pinout de la FPGA.
